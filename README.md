# NatureLens AI

ต้นแบบเว็บแดชบอร์ดสำหรับสำรวจโอกาสด้านการอนุรักษ์และฟื้นฟูธรรมชาติในประเทศไทย ผ่านแผนที่เชิงโต้ตอบและข้อมูลที่อธิบายอย่างเข้าใจง่าย

> สถานะ: Frontend prototype — ข้อมูล คะแนน กราฟ และ AI Insight ทั้งหมดเป็นข้อมูลจำลอง

## Features

- ค้นหาและเลือกพื้นที่ตัวอย่าง: เชียงใหม่ นครนายก และกระบี่
- Nature Opportunity Score และ Nature Profile
- กราฟดัชนีธรรมชาติย้อนหลัง 3 ปี
- แนวทางฟื้นฟูที่ปรับตามลักษณะของพื้นที่
- ชั้นข้อมูลสิ่งแวดล้อมบนแผนที่
- เปรียบเทียบพื้นที่และดูการเปลี่ยนแปลง
- สลับภาษาไทย/อังกฤษและส่งออกรายงานข้อความ
- รองรับหน้าจอเดสก์ท็อปและมือถือ

## Quick start

โครงการนี้ไม่มี dependency สำหรับหน้าเว็บ ต้องการเพียง Python 3 สำหรับเปิด local server:

```bash
python -m http.server 8000 --directory src
```

จากนั้นเปิด [http://localhost:8000](http://localhost:8000)

## Testing

ติดตั้ง Node.js 18 ขึ้นไป แล้วรัน:

```bash
npm test
```

## Project structure

```text
src/
  assets/       ภาพประกอบที่ใช้ในหน้าเว็บ
  index.html    โครงสร้างหน้า
  styles.css    รูปแบบและ responsive layout
  app.js        การโต้ตอบและการแสดงผล
  data.js       ข้อมูลพื้นที่จำลอง
tests/          automated tests
docs/           ข้อกำหนด การออกแบบ และแนวทางทดสอบ
```

## Data and map notice

- แผนที่เป็นภาพประกอบ ไม่ใช่แผนที่ขอบเขตการปกครองอย่างเป็นทางการ
- หมุดแสดงตำแหน่งจังหวัดโดยประมาณ ไม่ใช่พิกัด GIS ระดับสำรวจ
- ข้อมูลย้อนหลัง แหล่งข้อมูล และคำแนะนำเป็นตัวอย่างสำหรับสาธิตแนวคิด
- ระบบไม่เชื่อมต่อฐานข้อมูล แผนที่ออนไลน์ AI API หรือบริการเสียค่าใช้จ่าย
